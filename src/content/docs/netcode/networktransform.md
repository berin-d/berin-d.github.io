## Network Transform

> **Définition**
> Le composant NetworkTransform synchronise les mouvements (ou rotations) des objets du jeu sur le réseau.



#### Network Transform - Hiérarchie
----

```
(GameObject)
├── NetworkObject
├── NetworkTransform (pour synchroniser la position & rotation)
```
